GymFitness App

GymFitness is a Flutter-based mobile application designed to help users explore exercises and build better workout routines. The app provides detailed information about different muscle groups and related exercises using external fitness APIs.

The goal of the project is to create an interactive fitness guide where users can easily discover workouts for different parts of the body.

Features
- User authentication using Supabase
- Browse muscle groups
- View exercises related to each muscle
- Exercise details including equipment and instructions
- Clean and responsive mobile UI
- API integration for dynamic fitness data

Technologies Used
- Flutter (Mobile App Development)
- Dart
- Supabase (Authentication & Backend)
- REST APIs
- HTTP Package

App Workflow
- User opens the application
- User logs in or signs up
- User selects a muscle group
- App fetches exercises from the API
- Exercises are displayed with details and equipment information
